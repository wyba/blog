# 3.修改abiFilters

由于蓝畅的sdk用到so和科大讯飞用的有差异，所以我们需要加入科大讯飞的arm64-v8a，千万不要偷懒

修改app build.gradle配置

ndk {
            abiFilters "arm64-v8a", "armeabi-v7a"
        }

有了arm64-v8a和 armeabi-v7a，基本可以解决所有的arm适配问题
